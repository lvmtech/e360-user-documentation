# Debit / Credit Notes

The debit credit note tab in case modules show list of debit notes and credit notes issued for that case. As you can see in the following picture, there are five actions, "View Debit Note" or "View Credit Note", "View Audit History", "Edit", "Archive", "Delete" and "Export as Word", that can be done on each item of the list.
